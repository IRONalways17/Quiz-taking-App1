Tips for Creating Quizzes
  • Use clear, concise titles that describe what the quiz is testing
  • Provide detailed instructions in the description
  • Select an appropriate time limit based on the number and complexity of questions
  • After creating the quiz, you'll be able to add questions
  • If you set the quiz as inactive, students won't be able to see it until you activate it
Next Steps

After creating your quiz, you'll need to:

  1. Add questions to the quiz
  2. Set correct answers for each question
  3. Assign marks/points to each question
  4. Make sure the quiz is set to "Active" when you're ready for students to take it
